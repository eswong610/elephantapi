IDSP 2 2020

Api for an app to connect intellectually disabled children or young adults with educators who offer classes and social activities.

NestJS, Express, Nodejs, MongoDb

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
